本课作业要求

请参照右侧文档中的步骤，协同 AI 助手，在你的本地电脑桌面上双击运行这个网页计算器。成功后截图并点击下方按钮上传提交。

步骤 1：寻找你的 AI 搭档

点击网页版豆包、Kimi或风变AI，将以下指令粘贴发送给它：

“请帮我生成一个计算器网页代码，整体呈现精致美观的 iOS 风格。需要把所有的 HTML、CSS 和 JS 代码整理合并在一个单文件中。”
步骤 2：建立本地网页

在桌面新建空白文本文档，粘贴刚才复制的代码，并修改其文件名后缀，由 .txt 变为 .html。双击即可在本地打开使用！
